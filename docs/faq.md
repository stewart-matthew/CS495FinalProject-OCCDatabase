# Frequently Asked Questions

**Q1:** How is the user information being safely stored?

**A:** Using supabase's native authentication, contact information is restricted from non-authenticated individuals, and team member images are stored in private buckets.

**Q2:** Are there any likely roadblocks for installation / application of any technologies?

**A:** The website and the supabase database are both hosted online, and the project code is stored neatly in the team repository, so there should be no challenges in downloading any software.

**Q3:** How are users approved to have proper user/admin roles and the accompanying functionalities?

**A:** The sponsor in control of the database can assign user roles for team members with an admin flag, while the security policies and code provide adding, deleting, and editing functionality for admins for the different data types on their respective pages.
